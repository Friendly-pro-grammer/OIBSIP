# 🧮 Browser Calculator — WebDev Level 2

A sleek, fully functional browser-based calculator built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. Features a premium glassmorphism dark-mode UI with smooth micro-animations.

![Calculator Preview](https://img.shields.io/badge/Status-Complete-brightgreen?style=flat-square)  ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white)  ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)  ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Display Screen** | Shows the current input, full expression, and result |
| **Numeric Buttons** | Digits 0–9 and a decimal point button |
| **Operators** | Addition (+), Subtraction (−), Multiplication (×), Division (÷) |
| **Equals (=)** | Evaluates the expression and shows the result |
| **Clear (C)** | Resets the calculator to its initial state |
| **Backspace (⌫)** | Removes the last entered character |
| **Percent (%)** | Converts the current value to a percentage |
| **Division by Zero** | Displays a friendly error with a shake animation |
| **Operator Chaining** | Sequential operations without requiring a full reset |
| **Keyboard Support** | Full keyboard input (0–9, +, -, *, /, Enter, Backspace, Escape) |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup with accessible structure
- **CSS3** — Glassmorphism design, CSS Grid layout, custom properties, keyframe animations
- **JavaScript (Vanilla)** — Event delegation, no frameworks, no inline event handlers

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Friendly-pro-grammer/OIBSIP.git
   cd OIBSIP/WebDev-L2-Calculator
   ```

2. **Open in browser**
   - Simply open `index.html` in any modern browser, or
   - Use a local dev server:
     ```bash
     npx serve .
     ```

No build step, no dependencies — just open and go! 🎉

## 🎨 Design Highlights

- **Dark Glassmorphism** — Frosted-glass panel floating over animated gradient orbs
- **Micro-Animations** — Button press scale, display pop, error shake, ripple on click
- **CSS Grid** — Responsive 4-column button layout with spanning zero button
- **Custom Color Palette** — Curated violet/pink accent scheme for a premium feel
- **Google Fonts** — Uses *Inter* for crisp, modern typography
